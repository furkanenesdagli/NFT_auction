# Live auction 

Clone project
```bash
  git clone https://github.com/safalifter/live-auction-spring-react.git
```

Raise Containers
```bash
  docker-compose up
```

URL

`http://localhost:3000`

`http://localhost:8080/swagger-ui/index.html`
